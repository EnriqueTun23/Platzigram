Platzigram
=============

App sencilla estilo instagram


## Instalacion
### Primer paso:
Haber instalado virtualenv

### Crear un virtualenv

`virtualenv -p /usr/bin/python3 env`

### Arrancar la maquina 

`source env/bin/activate`

#### ---ojo---
instalar los paquetes de requirements.txt


` pip install -r requirements.txt `
